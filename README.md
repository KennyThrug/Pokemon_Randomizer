# Pokemon_Randomizer
Launcher (and possibly runner) for a randomizer pokemon

## Features:

- [ ] Launcher to set settings
- [ ] Automatic Compilation of Roms
- [ ] Up to Generation 8 pokemon
- [ ] Physical Special Split Implemented
